How does javascript work on browser:

জাভাস্ক্রিপ্ট প্রোগাম ব্রাউজারে যখন রান করা হয় ব্রাউজার ইঞ্জিন কোডটিকে রিসিভ করে আউটপুট অউটপুট দেওয়া পর্যন্ত মোটামুটি চারটি ধাপে কাজ সম্পন্ন করে :

  • Parser : এই ধাপে সোর্স কোডটিতে syntactic কোন ভুল আছে কিনা চেক করা হয়। ভুল থাকলে এখানেই এক্সিকিউশন বন্ধ করে দেয়।
  • AST (Abstract Syntax Tree) : যদি Parser এ কোন ভুল না ধরা পরে তাহলে AST নামের একটি ডাটা স্ট্রাকচার তৈরি হয়।
  • Conversion to Machine Code : AST তৈরি করার পর ব্রাউজার ইঞ্জিন সোর্স কোডকে JIT ( Just in Time ) কম্পাইলারের মাধ্যেম কম্পাইল করে এবং মেশিন কোডে রূপান্তরিত করে।
  • Machine Code : সবশেষে ইঞ্জিন এই মেশিন কোডকে রান করে ফাইনাল অউটপুট দেয়।

    • Event Lopp:

    জাভাস্ক্রিপ্ট ব্রাউজারে রানটাইমে কোড গুলো লুপ আকারে এক্সিকিউট হয়। একে Even Loop বলা হয়।
    সংক্ষেপে বলা যায় : জাভাস্ক্রিপ্ট কোডের কিছু অংশ Synchronous ভাবে আর কিছু অংশ Asynchronous ভাবে রান হয়। Synchronous অংশ গুলো উপর থেকে Stack আকারে রান হয় এবং এর পরে Asynchronous গুলো Queue আকারে রান হয়।আবার asynchronous গুলোকে পর্যায়ক্রমে নির্দিষ্ট সময় পরে এক বা একাধিকবারও রান করানো যেতে পারে। অবজেক্ট গুলা Heap নামে মেমোরি অ্যালোকেশন করে (Unstructured) । অপ্রয়োজনীয় অংশ গুলো garbage collection এ চলে যায়। এই পুরো বিষয়কেই জাভাস্ক্রিপ্টের Event Loop বলা হয় ।
